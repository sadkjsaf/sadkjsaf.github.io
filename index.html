Linux学习日记006–cd命令学习

1.前言

今天我们见解,cd

cd 命令是命令 change directory 的简写，cd 命令用于改变当前的工作目录的命令，命令所在的路径为 /bin/cd。

2.常用参数与使用场景：

2.1.1 切换到指定目录

cd 命令可以通过指定目录路径将当前目录切换到指定目录

2.1.2 返回上一个目录

cd 可以返回上一个目录

2.1.3 返回上级目录

cd 命令可以通过 .. 返回当前目录的上级目录

2.1.5 把上个命令的参数作为cd参数使用

cd 命令可以通过 !$ 把上个命令的参数作为 cd 参数使用

3.今日学习总结：

cd命令是在终端中切换当前工作目录的命令。它允许用户进入指定的目录并在其中浏览和执行操作。
